# Security Policy

## Reporting a Vulnerability

If you discover a security issue related to this repository — including
sensitive data exposure, misconfigured examples, or content that could
enable harm — please report it responsibly.

**Do not open a public issue.**

Report privately via GitHub's Security Advisory system:
[Report a vulnerability](https://github.com/commit-issues/google-dorking/security/advisories/new)

## What to Include

- Description of the issue
- File(s) affected
- Any suggested remediation

## Response

Reports will be reviewed and addressed as quickly as possible.
This is a solo-maintained project — please allow reasonable response time.

## Scope

This repository contains **educational reference material only**.
No credentials, keys, tokens, or personally identifiable information
should exist anywhere in this repository. If you find any, please
report it immediately.
